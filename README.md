

### <SNS 제작 프로젝트>

#### **프로젝트 소개**

####  **목적**
- 수업 시간에 배운 crud(create/read/update/delete) 실습. crud를 활용한 피드 관리 시스템 구축

####  **구조**
- 회원가입 / 로그인 / 피드 / 검색 / 피드 추가 / 프로필 수정 

####  **기능**
- MySql과 서버를 연결하여 각 회원 계정에서 피드 관리 시스템의 조회, 수정, 삭제, 생성 등을 구현하였습니다.
